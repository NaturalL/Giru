功能

错误日志上报: 目前支持 Logback, Log4j 1.x, 自动上报 ERROR 及以上级别.
告警: 按异常stackTrace聚合,提供新异常和突增异常告警


接入:

添加依赖

    <dependency>
        <groupId>com.biasee.giru</groupId>
        <artifactId>giru-spring-boot-starter</artifactId>
        <version>{{version}}</version>
    </dependency>


客户端配置:


    giru.event.url #服务端地址，格式如：http://localhost:8092
    spring.application.name #应用名
    giru.event.error-log.enabled #启用错误日志采集,默认 true
    giru.event.report.enabled #启用事件上报,默认 true
    giru.event.error-log.stacktraceDepth #Exception栈深度,默认50
